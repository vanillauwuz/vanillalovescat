Calculate Time 
โปรแกรมนี้เป็นโปรแกรมที่ใช้ให้ผู้ใช้ป้อนจำนวนนาทีทั้งหมดแล้วแปลงเป็น
หน่วยเวลาเป็นเดือน วัน ชั่วโมง และนาที

ส่วนประกอบของโปรแกรม:

ไลบรารี (Library) ที่ใช้งาน: stdio.h

ตัวแปรที่ใช้งาน:

1. totalMinutes: ตัวแปรชนิด int เพื่อเก็บค่าจำนวนนาทีทั้งหมดที่ผู้ใช้ป้อน
2. months: ตัวแปรชนิด int เพื่อเก็บค่าจำนวนเดือนที่คำนวณได้
3. days: ตัวแปรชนิด int เพื่อเก็บค่าจำนวนวันที่คำนวณได้
4. hours: ตัวแปรชนิด int เพื่อเก็บค่าจำนวนชั่วโมงที่คำนวณได้
5. minutes: ตัวแปรชนิด int เพื่อเก็บค่าจำนวนนาทีที่คำนวณได้

โค้ดที่ใช้งาน:

1. printf(): ฟังก์ชันสำหรับแสดงผลข้อความหรือค่าตัวแปรในหน้าจอคอนโซล
2. scanf(): ฟังก์ชันสำหรับรับค่าที่ผู้ใช้ป้อนเข้ามา
3. ตัวดำเนินการทางคณิตศาสตร์: ใช้ในการคำนวณหา
จำนวนเดือน วัน ชั่วโมง และนาทีจากจำนวนนาทีทั้งหมด

วิธีใช้ตัวดำเนินการทางคณิตศาสตร์เพื่อแบ่งจำนวนนาทีทั้งหมดออกเป็นหน่วยเวลาต่างๆ

months: จะมีการคำนวณและกำหนดค่าตัวแปร months 
โดยนำจำนวนนาทีทั้งหมดของ (totalMinutes) หารด้วยผลคูณของ 
60 (นาที)x 24 (ชั่วโมง)x 30 (วัน) เพื่อหาจำนวนเดือนที่คำนวณได้

days: จะมีการคำนวณและกำหนดค่าตัวแปร days โดยใช้เศษจากขั้นตอนก่อนหน้านี้ 
และหารด้วยผลคูณของ 60 (นาที)x 24 (ชั่วโมง) เพื่อหาจำนวนวันที่คำนวณได้

hours: จะมีการคำนวณและกำหนดค่าตัวแปร hours โดยใช้เศษจากขั้นตอนก่อนหน้านี้ 
และหารด้วย 60 (นาที) เพื่อหาจำนวนชั่วโมงที่คำนวณได้

minutes: จะมีการคำนวณและกำหนดค่าตัวแปร minutes โดยใช้เศษจากขั้นตอนก่อนหน้านี้ 
หารด้วย 60 (นาที) เพื่อหาจำนวนนาทีที่คำนวณได้

#include <stdio.h>

int main() {
    int totalMinutes;
    printf("Enter the total number of minutes: ");
    scanf("%d", &totalMinutes);

    int months = totalMinutes / (60 * 24 * 30);
    int days = (totalMinutes % (60 * 24 * 30)) / (60 * 24);
    int hours = (totalMinutes % (60 * 24)) / 60;
    int minutes = totalMinutes % 60;

    printf("Months: %d\n", months);
    printf("Days: %d\n", days);
    printf("Hours: %d\n", hours);
    printf("Minutes: %d\n", minutes);

    return 0;
}
