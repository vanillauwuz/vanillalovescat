Scanf 
โปรแกรมนี้เป็นโปรแกรมที่ใช้ให้ผู้ใช้ป้อนเลขจำนวนเต็มสองจำนวนแล้วแสดงผลลัพธ์ของ
การบวก ลบ คูณ และหารของตัวเลขทั้งสอง

ส่วนประกอบของโปรแกรม:

ไลบรารี (Library) ที่ใช้งาน: stdio.h

ตัวแปรที่ใช้งาน:

1. num_int1: ตัวแปรชนิด int เพื่อเก็บค่าเลขจำนวนเต็มที่ผู้ใช้ป้อน
2. num_int2: ตัวแปรชนิด int เพื่อเก็บค่าเลขจำนวนเต็มที่ผู้ใช้ป้อน

โค้ดที่ใช้งาน:

1. printf(): ฟังก์ชันสำหรับแสดงผลข้อความหรือค่าตัวแปรในหน้าจอคอนโซล
2. scanf(): ฟังก์ชันสำหรับรับค่าที่ผู้ใช้ป้อนเข้ามา
3. ตัวดำเนินการบวก (+), ลบ (-), คูณ (*), และหาร (/)
เพื่อใช้ในการดำเนินการทางคณิตศาสตร์บนตัวแปรชนิด int และ float
#include <stdio.h>

int main() {
    int num_int1, num_int2;
    printf("Enter the first number: ");
    scanf("%d", &num_int1);
    printf("Enter the second number: ");
    scanf("%d", &num_int2);

    printf("%d + %d = %d\n", num_int1, num_int2, num_int1 + num_int2);
    printf("%d - %d = %d\n", num_int1, num_int2, num_int1 - num_int2);
    printf("%d * %d = %d\n", num_int1, num_int2, num_int1 * num_int2);
    printf("%d / %d = %.3f\n", num_int1, num_int2, (float)num_int1 / num_int2);

    return 0;
}
